035 Async, defer, динамические скрипты

Атрибуты скриптов:

  • script defer src='' - "отложенный скрипт"

    Выполняется после загрузки DOM структуры (загрузка в фоновом режиме)

  • script async src='' - "ни кого не ждет скрипт"

    Загружается в фоновом режиме и выполняется сразу после загрузки)
